## Описание
Позволяет осуществить проверку стоимости акций и фондов и актуализировать данные в базе данных Notion.
## Примечание
Для подключение к API Notion необходим secret token, а также настроенная интеграция с базой. См. [Notion API docs](https://developers.notion.com/reference/intro)

Данные берутся с https://www.tinkoff.ru/invest/

Обязательные поля БД:
- Наличие числового поля "Current Price";
- Наличие поля с датой актуализации "Update date";
- Наличие текстового поля "Kind", которое может содержать"STOCKS" (акции) или "ETFS" (фонды);
- Наличие текстового поля "Ticker", содержащего актуальный тикер.
